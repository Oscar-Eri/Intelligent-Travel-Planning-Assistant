## 🚀 快速开始

### 启动后端
```bash
cd backendtest
python main.py
```

或双击：`backend\启动服务.bat`

### 启动前端
```bash
cd frontendtest
npm run dev
```

### 一键启动
双击根目录：`启动全部服务.bat`

## 📝 开发提示

- **新增功能** → backend/services/
- **新增API** → backend/routes/
- **测试代码** → backend/tests/
- **前端组件** → frontend/src/app/components/
